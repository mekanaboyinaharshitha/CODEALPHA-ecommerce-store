# CodeAlpha E-Commerce Book Store 📚

## 🚀 Project Description
A simple full-stack e-commerce web application built using HTML, CSS, JavaScript, and Node.js.

## ✨ Features
- User Registration & Login
- Product Listing Page
- Shopping Cart System
- Order Placement API
- LocalStorage Cart
- Simple Backend (Express.js)

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express.js

## 📦 How to Run
1. Clone repository
2. Install dependencies:
   npm install
3. Run server:
   node server.js
4. Open:
   http://localhost:5000
